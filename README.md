# d-server
Learning more about MEAN.

### Running
1. Clone the repository
```
$ git clone https://github.com/dannydenenberg/d-server.git
```
2. cd into the folder and install the packages
```
$ cd d-server
$ npm install
```
3. Run the project
```
$ npm start
```
4. Open your web browser and input:
```
http://localhost:3000
```
